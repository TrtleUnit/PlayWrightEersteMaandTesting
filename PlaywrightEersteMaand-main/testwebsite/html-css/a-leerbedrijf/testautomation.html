Een van onze geautomatiseerde tests

Ons team schrijft computerscripts om websites geautomatiseerd te testen. Alle handelingen die een mens met een browser kan uitvoeren, scripten wij. Dit is zo'n test. Het script gebruikt de functionaliteiten van de website zoals een echte gebruiker dat ook zou doen.

In de jaren '90 werden websites handmatig getest. Automatisering van het testen kwam pas na 2000 op gang. Handmatig testen heeft nog steeds een rol in de ICT maar geautomatiseerd testen vormt nu de bulk van het testwerk.

De code wordt in de video twee gerund. De eerste keer zie de code tijdens de run en in de twee run wordt de test in de browser getoond. Tijdens de uitvoering van het script wordt voortdurend getoetst of de website correct reageert en of de gegevens in de browser correct zijn.
