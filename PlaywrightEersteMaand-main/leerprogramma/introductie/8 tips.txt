TIPS

------------------------------------------------------------------------

In de ICT draait het om efficientie. Je zult efficient met je digitale 
werkomgeving moeten kunnen omgaan. Je moet handig zijn in het gebruik
van je digitale tools (toolskills).

1. Organiseer je digitale werkomgeving. Maak een apart leer/werk account op windows 
aan voor je leer/werkomgeving. 

2. Creeer vaste plekken voor de dingen die je in je leer/werkaccount opslaat
zodat je snel kunt vinden waar iets staat. Geef de dingen die je het vaakst 
gebruikt een plek op het hoogste niveau.  

3. Maak in je browser een aparte map aan voor je leerbronnen (zie file 
links.txt in de map tools).

4. Zorg dat je per onderwerp twee of drie go-to sites hebt waar je 
terecht kunt als je op zoek naar uitleg bent. Bookmark die sites en beheer die
bookmarks. 

5. Hou overzicht als je werkt. Open geen 3 browservensters met ieder 6 tabs 
waar een webpagina op staat. 

6. Zorg dat je handig bent met 
- vs-code,
- de windows verkenner (explorer),
- de instellingen van windows,
- windows taakbeheer(Task Manager) ,
- outlook,
- discord.

7. Zet windows one-drive uit. 

8. Ga geen zogenaamd handige extensies in vs-code of op je browser installeren. 

9. Verzin zelf ook eens een tip.

10. Je mag dit leerprogramma gebruiken zoals je wil. Het hoeft niet maar je mag
aantekeningen maken in files als je dat helpt om te leren. Schrijf in scriptfiles 
comments bij code zodat je ook een volgende keer als je langs komt begrijpt wat 
er gebeurt. 

Je kunt de testwebsite uitbreiden als je dat wil. Dan maak je er bijv. een groep links 
bij die naar je eigen pagina's gaan. Bijvoorbeeld je cv pagina. Voor die eigen 
pagina's kun je weer eigen tests schrijven. 

Je hebt overal werkende voorbeelden. Ga dingen uitproberen. Dat is de manier 
waarop je leert. Dat is de manier om te leren werken met de tools maar ook met
code.

11. Het begin is zwaar. Je zult serieus tijd moeten investeren. Het opbouwen van
ICT-kennis is als een vliegwiel op gang brengen. Het kost inspanning om de eerste
omwentelingen te maken maar je gaat ontdekken dat als het eenmaal een beetje 
draait het steeds minder moeite kost om het wiel sneller te laten draaien.

12. Als dat wiel na 6 maanden al een beetje draait dan heb je het goed gedaan en
kun je in principe zonder veel aansturing je studie zelf vervolgen zolang je het 
wiel draaiend houdt. 

13. Zes maanden zijn zo voorbij. Benut de tijd in het project om zoveel mogelijk
te leren. Weet je even niet wat je doen moet? Verken de opties van vs-code, van 
de devtools van je browser, verken playwright, typescript, discord, windows. Verken
het vakgebied ICT, verken software testen. Bouw je handigheid met de tools op in 
verloren momenten of als je je verveelt. 

Verken de arbeidsmarkt, bekijk de vacatures op de relevante sites. Vraag je zelf af 
waaraan je nog kunt werken om verder te komen. Start een portfolio project.
