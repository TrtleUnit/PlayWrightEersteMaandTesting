Een video over een voorbeeld portfolioproject.

Stapsgewijze instructie

1. Start Visual Studio op.
2. In het scherm 'Create a new project' type in de tekstbox 'Windows.forms c#'. // een windows form (ook winform genoemd is een app op windows)
3. Kies de top optie. // download de template voor een desktop app
4. In het scherm 'Configure your new project' type de naam van je project.
5. Klik 'next'.
6. In scherm 'Additional information' zorg dat .NET 8.0 gekozen is en klik 'Create'. // Zorgt ervoor dat je app ook voor de toekomst met windows geintegreerd blijft.
7. Wacht tot je 'Form1' ziet verschijnen. Dat is de GUI die je hebt gecreeerd. // De GUI is je op het beeldscherm van je computer als je de app op windows runt.
8. Ga naar het topmenu van vs-code dat met 'File Edit View' begint. In dat rijtje staat 'Tools'.
9. Onder 'Tools' zoek 'NugetPackage Manager' en daaronder staat een submenu 'Manage NugetPackages for solution'.
10. Klik 'Manage NugetPackages for solution'.
11. Een nieuw scherm opent in vs, In het nieuwe scherm zie je bovenin 'Browse Installed Updates Consolidate'.
12. Klik op 'Browse' en type in de textbox 'webview2 winforms'.
13. Kies de optie 'Microsoft.Web.Webviews' (de top optie).
14. Een nieuwe kolom wordt zichtbaar rechts en bovenaan staat groot 'Microsoft.Web.Webviews'..
15. Klik hier 'Project'. Dat ook het vakje eronder aan wordt gevinkt is prima.
16. Klik button met tekst 'install'.
17. In het pop-up screen klik 'Apply'. // je hebt nu de library voor de browser gedownload in je app.
18. Sluit het scherm 'Nuget - Solution' en ga naar 'Form1.cs[Design]' terug. // De functionaliteit die je ziet als je op 'Form1.cs[Design]' noemen we de designer.
19. Naast de designer zie je aan de linkerkant het opschrift 'Toolbox'. Klik daarop. De Toolbox opent.
20. Klik binnen de Toolbox bovenin onder 'Pointer' op 'Webview2(Miscrosoft etc)'.
21. Klik op de designer. De Toolbox sluit nu en je hebt een webview2 in je designer.
22. Maak die groter. Volg de video.

En dit kan het resultaat worden.
