Over deze site

De site dient diverse doeleinden:
  • Een digitaal visitekaartje met relevante en representatieve content voor het leerbedrijf Bee LKQ
  • Een template voor portfolioprojecten
  • Een testwebsite om testsautomatisering te trainen

Digitaal visitekaartje

  • Voor de ketenpartner: zoek je een ketenpartner die je client verder kan helpen?
  • Voor de potentiele deelnemer: wil je het vak van software tester verkennen?
  • Voor de deelnemer: wil je je als tester verder ontwikkelen? Er staat heel veel relevante content op de site.

Template

  • De site is gebouwd rondom 5 verschillende thema's: Leerbedrijf, Educatief, Baanperspectief, Portfolio en Templates. Aan die kapstok is alles opgehangen. Deze indeling zorgt voor een eenvoudige maar doeltreffende structuur die ook kan worden toegepast in andere projecten.
  • Je kunt met de werkende website als voorbeeld een mooi portfolio project voor je cv bouwen.

Testwebsite

  • Een kopie van de website is geintegreerd in de door Bee ontwikkelde, digitale leeromgeving PlaywrightEersteMaand die gebruikt wordt voor de onboarding van nieuwe deelnemers. In deze leeromgeving is ook de software geintegreerd waarmee websites geautomatiseerd getest kunnen worden (playwright).
